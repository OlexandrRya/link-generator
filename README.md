## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OlexandrRya/link-generator.git
   cd link-generator
   ```

2. Build the project:
   ```bash
   make build
   ```

3. Start the services and wait for the containers to be ready:
   ```bash
   make up
   ```

4. Run the development server:
   ```bash
   make serve
   ```
